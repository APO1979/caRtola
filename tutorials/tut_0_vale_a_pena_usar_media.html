E a m√©dia, vale a pena?

Vamos agora ao que interessa, comparar se a m√©dia √© um bom indicador (modelo) para prever o desempenho dos jogadores.

Como fazemos isso? O Erro M√©dio Quadr√°tico (RMSE, do ingl√™s, Root Mean Square Error) e Erro Absoluto da M√©dia (MAE, do ingl√™s, Mean Absolute Error) s√£o m√©tricas usadas para avaliar modelos de predi√ß√£o com desfecho num√©rico - e.g., pontua√ß√£o.

Vamos, ent√£o avaliar o qu√£o bem a m√©dia √© capaz de prever os resultados da nova rodada. Antes disso, precisamos preparar os dados.

                group_by(atletas.atleta_id) %>%
                summarise(pontos = mean(atletas.pontos_num))

Agora podemos ver como a m√©dia est√° ligada aos resultados da rodada 9. Para isso, vamos ao pr√≥ximo gr√°fico.

    facet_wrap(~ jogos) +      
    theme_hc() + scale_colour_hc() +
    labs(title = "Dispers√£o da m√©dia e pontua√ß√£o", x = "Pontos", y = "M√©dia")

RMSE e MAE

dados <- mutate(dados, erro_media = media - pontos, erro_mediana = mediana - pontos)

# Fun√ß√£o Root Mean Squared Error
RMSE <- function(error)
{
    sqrt(mean(error^2, na.rm = TRUE))
}
 
# Fun√ß√£o Mean Absolute Error
MAE <- function(error)
{
    mean(abs(error), na.rm = TRUE)
}

# Comparar resultados da m√©dia e mediana
# RMSE
print(sapply(dados[, c("erro_media","erro_mediana")], RMSE))
##   erro_media erro_mediana 
##     5.037917     5.026678
# MAE
print(sapply(dados[, c("erro_media","erro_mediana")], MAE))
##   erro_media erro_mediana 
##     3.607697     3.478681

Conclus√£o

Sim, ao comparar a m√©dia com a mediana, podemos notar que em m√©dia erramos cinco pontos para mais ou menos a pontua√ß√£o de um dado jogador. Isto significa que um jogador que pontua√ß√£o m√©dia na oitava rodada igual a 10 pontos, provavelmente pontuar√° entre 5 e 15 pontos. Assim, fazer predi√ß√µes com base em m√©dias parece ser algo arriscado, embora melhor que simplesmete chutar um atleta.
